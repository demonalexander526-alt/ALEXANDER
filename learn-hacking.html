Hacking Topics & Tutorials

WiFi Hacking

Learn wireless network security, including WPA/WPA2/WPA3 cracking, deauthentication attacks, and rogue access point detection. Understand the fundamentals of wireless security protocols and common vulnerabilities.

Key concepts: Handshake capture, dictionary attacks, PMKID attacks, and wireless reconnaissance.

Basic Commands

iwconfig airmon-ng start wlan0 airodump-ng mon0 airodump-ng -c 6 --bssid XX:XX:XX:XX:XX:XX -w capture mon0 aireplay-ng --deauth 10 -a XX:XX:XX:XX:XX:XX mon0 aircrack-ng -w wordlist.txt capture-01.cap

Wireshark

Become proficient in network protocol analysis using Wireshark. Learn packet capture, filtering, analysis of common protocols (HTTP, DNS, TCP/IP), and identifying security issues in network traffic.

Topics: Display filters, capture filters, protocol dissection, and traffic analysis for troubleshooting and security auditing.

Usage Examples

wireshark tshark -i eth0 -w capture.pcap wireshark capture.pcap tcpdump -i eth0 -w capture.pcap

Ferobuster (Directory Busting)

Learn directory and file enumeration techniques using Ferobuster. Discover hidden directories, files, and endpoints on web servers through brute-force and dictionary-based attacks.

Covers: Wordlist selection, extension testing, recursive scanning, and filtering techniques for efficient enumeration.

Basic Usage

feroxbuster -u http://example.com -w /path/to/wordlist.txt feroxbuster -u http://example.com -w wordlist.txt -x php,txt,html feroxbuster -u http://example.com -w wordlist.txt --recursive

Fernet (Cryptography)

Master symmetric encryption using Python's Fernet library. Learn secure key generation, encryption/decryption of data, and implementing cryptography in applications for data protection.

Topics: Key management, token generation, secure communication, and cryptographic best practices.

Python Examples

from cryptography.fernet import Fernet key = Fernet.generate_key() cipher = Fernet(key) encrypted = cipher.encrypt(b"secret message") decrypted = cipher.decrypt(encrypted)

TC (Traffic Control)

Learn Linux Traffic Control (tc) for network shaping, QoS, and bandwidth management. Understand queuing disciplines, filters, and advanced networking configurations for system administration and security testing.

Covers: Bandwidth limiting, traffic prioritization, delay simulation, and network testing scenarios.

Essential Commands

tc qdisc add dev eth0 root netem delay 100ms tc qdisc add dev eth0 root tbf rate 1mbit burst 32kbit latency 400ms tc qdisc show dev eth0 tc qdisc del dev eth0 root

Nmap (Network Mapper)

Master the essential network scanning tool for discovering hosts, services, and vulnerabilities. Learn port scanning, service detection, OS fingerprinting, and script-based vulnerability assessment.

Key features: Host discovery, port scanning, version detection, OS detection, and NSE scripts for advanced scanning.

Basic Commands

nmap -sn 192.168.1.0/24 nmap -sV -O 192.168.1.100 nmap -p 1-65535 --open 192.168.1.100 nmap --script vuln 192.168.1.100

Burp Suite

Learn web application security testing with Burp Suite. Master intercepting proxy, scanner, intruder, repeater, and decoder tools for comprehensive web vulnerability assessment.

Covers: Proxy interception, automated scanning, manual testing, session handling, and extension development.

Usage

burpsuite burpsuite --headless --scan-url=http://example.com

SQLMap

Features: Database enumeration, data dumping, shell access, and advanced injection techniques.

Basic Commands

sqlmap -u "http://example.com?id=1" --dbs sqlmap -u "http://example.com?id=1" -D database --tables sqlmap -u "http://example.com?id=1" -D database -T table --dump sqlmap -u "http://example.com?id=1" --os-shell

Hashcat

Advanced password cracking tool supporting multiple hash types and attack modes. Learn dictionary attacks, brute force, mask attacks, and GPU-accelerated cracking techniques.

Supports: MD5, SHA, NTLM, WPA, and thousands of other hash formats with optimized algorithms.

Basic Commands

hashcat -m 0 -a 0 hashes.txt wordlist.txt hashcat -m 1000 -a 3 ntlm_hashes.txt ?a?a?a?a?a?a hashcat -m 2500 -a 0 wpa_handshake.hccapx wordlist.txt hashcat --show hashes.txt

Hydra

Fast network logon cracker supporting numerous protocols. Learn brute force and dictionary attacks against login systems, including SSH, FTP, HTTP, and database services.

Features: Parallel connections, different attack modes, and support for custom modules.

Basic Commands

hydra -l admin -P passwords.txt ssh://192.168.1.100 hydra -L users.txt -P pass.txt ftp://192.168.1.100 hydra -l admin -P rockyou.txt http-post-form "/login:username=^USER^&password=^PASS^:Invalid"

Nikto

Web server scanner that performs comprehensive tests against web servers for multiple items. Detect outdated software, configuration issues, and potential security vulnerabilities.

Covers: Server configuration, outdated components, and common web server vulnerabilities.

Basic Commands

Volatility

Advanced memory forensics framework for analyzing RAM dumps. Extract processes, network connections, passwords, and malware from memory images for incident response and malware analysis.

Features: Process analysis, network artifacts, registry analysis, and plugin-based architecture.

Basic Commands

John the Ripper

Features: Support for hundreds of hash types, optimized for CPU cracking, and extensive customization options.

Basic Commands
