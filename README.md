## 🚀 How to Use

1. **Open the Portfolio**: Open `index.html` in any modern web browser
2. **Toggle Dark Mode**: Click the moon/sun icon in the top right of the navbar to switch between light and dark modes
   - Moon icon = Light mode is active (click to switch to dark)
   - Sun icon = Dark mode is active (click to switch to light)
3. **Navigate**: Use the navigation menu to explore different sections
4. **Main Pages**:
   - Click "Programming" to view detailed programming language expertise
   - Click "Hacking" to explore cybersecurity knowledge
   - Use smooth scroll navigation to jump between sections

**Note**: Your theme preference is automatically saved and will persist when you revisit the page!
